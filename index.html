The Legend of Zelda (1986)

This is a replica of the 1986 game "The Legend of Zelda" focusing on the first dungeon level. It also comes with a custom level featuring a new mechanic, which can be accessed by pressing "4." Controls include the arrow keys or WASD to move, X to use the sword, Z to use the secondary attack, and spacebar to shuffle between secondary weapons. 1 can also be pressed to turn on godmode.

This project features C# and Unity. Play it here!

Chess 2 Electric Boogaloo

This is a rapid prototype of a Chess-based roguelike that uses procedural generation to create the level and has movement and enemy attacks based on chess rules. Score can be improved by capturing pieces and the game is beat by reaching the final room.

This project features C# and Unity. Play it here!
